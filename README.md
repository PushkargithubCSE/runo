# ⚡ Runo

> Run any project instantly — zero setup.

Runo is a lightweight CLI tool that automatically detects, installs, and runs your project with a single command.

No manual setup. No dependency headaches.

---

## 🚀 Features

- 🔍 Auto-detects project type (Python / Node)
- 📦 Installs dependencies automatically
- ▶ Runs your project instantly
- ⚡ Zero setup experience
- 🧠 Simple and extensible architecture
